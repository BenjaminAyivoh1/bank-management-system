# Bank Management System

A GUI-based Bank Management System built with Python and Tkinter.

## Features

- Create bank accounts
- Deposit money
- Withdraw money using PIN verification
- View transaction history
- Real-time balance updates
- Automatically generated account numbers
- Show/Hide account number feature
- Input validation and error handling
- Clean graphical user interface (GUI)

## Technologies Used

- Python
- Tkinter
- Object-Oriented Programming (OOP)

## Concepts Demonstrated

- Encapsulation
- Classes and Objects
- Methods
- GUI Development
- Event Handling
- Error Handling
- Transaction Management
